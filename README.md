# anan-amount

一个基于微信小程序云开发模板继续改造的练习仓库。

目前仓库里已经包含了基础的小程序页面结构，并在此之上增加了商品、登录、注册、个人中心等页面，适合作为学习微信小程序界面搭建、交互组织和云开发项目结构的个人实验仓库。

## 当前内容

- 微信小程序工程配置 `project.config.json`
- 小程序主配置 `miniprogram/app.json`
- 已存在页面：
  - `pages/index/index`
  - `pages/example/index`
  - `pages/shop/index`
  - `pages/register/index`
  - `pages/login/index`
  - `pages/profile/index`
- 云开发相关目录结构

## 适合用来做什么

- 练习微信小程序页面开发
- 继续补全商品展示、登录注册、个人中心等业务流程
- 接入云函数、数据库和文件存储
- 作为后续完整小程序项目的原型仓库

## 本地运行

1. 克隆仓库到本地。
2. 使用微信开发者工具打开仓库根目录。
3. 确认小程序环境、AppID 和云开发环境配置。
4. 从 `miniprogram/` 目录开始继续开发和调试。
